# Data Sources

## Ocean County Parcel Geodatabase
- Contains parcel geometry for Toms River.
- Used for parcel boundaries and township clipping.

## MOD-IV Assessment Data
- Contains tax, sales, and building-related attributes.
- Used for parcel-level property assessment context.

## FEMA Flood Hazard Data
- Contains FEMA flood zone polygons and base flood elevation fields.
- Used to identify mapped flood hazard areas in Toms River.
